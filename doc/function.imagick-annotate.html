imagick_annotate --  Annotates an image with a text

Description

bool imagick_annotate (resource imagick_handle, array options)

Warning

This function is EXPERIMENTAL. That means, that the behaviour of this function, this function name, in concreto ANYTHING documented here can change in a future release of PHP WITHOUT NOTICE. Be warned, and use this function at your own risk.

This function can be used to draw a text on an image. See the example.

Example 1. imagick_annotate() example

imagick_annotate($handle,array(
            "primitive"=>"text 150,150 hello world",
            "pointsize"=>60,
            "antialias"=>1,
            "stroke"=>'green',            
            "fill"=>'#ff7755',                        
            "font"=>"Arial.ttf",
            "rotate"=>90
            ));
?>
