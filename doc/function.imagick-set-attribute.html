(unknown)

imagick_set_attribute --  Sets an image attribute.

Description

bool imagick_sample (resource imagick_handle, mixed attribute [, string attribute_value])

Warning

This function is EXPERIMENTAL. That means, that the behaviour of this function, this function name, in concreto ANYTHING documented here can change in a future release of PHP WITHOUT NOTICE. Be warned, and use this function at your own risk.

This function takes either two string with the attribute and the value, or a array with attributes as key and the values as values...

At the moment following attributes are supported:

adjoin
delay
format
magick
quality
size

More Options will follow soon, it's more or less only copy&paste work.

Note: For a detailed list of what the attributes are, see http://www.imagemagick.org/www/perl.html#seta

Example 1. imagick_set_attribue() example

<?php
$handle = imagick_create ()
    or die ("Could not create handle");
imagick_read($handle,"mypic.gif");
imagick_set_attribue($handle,"quality",10);
imagick_write($handle,"mypic.jpg");
imagick_free($handle);
?>

Example 2. imagick_set_attribue() example with more than one attribute at once

<?php
$handle = imagick_create ()
    or die ("Could not create handle");
imagick_read($handle,"mypic.gif");
imagick_set_attribue($handle,array("quality"=>10,"format"=>"jpeg");
imagick_write($handle,"mypic.jpg");
imagick_free($handle);
?>
